PDF.js Mini Viewers Demo

This is a simple demonstration of PDF.js Mini Viewers (PMV), it should look great on mobile devices, tablets, and laptop or desktop computers! For installation and configuration help please see the PDF.js Mini Viewers (PMV) GitHub repository. PMV is an open source community supported project, if you would like to help please consider tackling an issue or making a donation to keep the project alive.
